Some time ago, I released my application <a target="_blank" href="https://market.android.com/details?id=com.esperanto">Intense Esperanto</a>. The learning method used in I EO has been proposed by <a target="_blank" href="http://www.chucksmith.de/">Chuck Smith</a> and Judith Meyer, and I developed the application in collaboration with them. Well, now I''m releasing more versions for more languages: this time, <a target="_blank" href="https://market.android.com/details?id=com.german">Intense German</a> and <a target="_blank" href="https://market.android.com/details?id=com.latin">Intense Latin</a>. Those last versions are not free, but I guess the price is fair: it give me some extra-money to continue developing more applications, while Intense Esperanto and the others applications remain free.
