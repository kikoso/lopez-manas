TrafficVision is an application presented in the context of <a target="_blank" href="http://abredatos.es/">AbreDatos</a> (the Spanish Version of <a target="_blank" href="http://sunlightlabs.com/contests/appsforamerica/">Apps for America</a>). With the popularization of traffic cameras on our roads and the improvement of computer vision techniques, new possibilities for improving traffic management are arising. trafiCVision attempts to demonstrate this concept by joining these two worlds.
<br/>
So we've developed a platform where camera images of traffic from the Dirección General de Tráfico (National Traffic Department) scattered throughout the Spanish territory are converted into numeric information displaying the number of vehicles each processed image contains. Thus, an interactive map is generated which graphically reflects the traffic congestion in the area where the camera is located. This map provides information in a way that would not be possible just by visiting the image.

The application can be accessed <a target="_blank" href="http://www.trafficvision.net/">here</a>. The source code can be checked <a target="_blank" href="http://code.google.com/p/traffic-vision">here</a>.
